Tsunami Detection Simulation

A simulator which allows users to input values and produces a log text file consisting of the results of the simulation. This project was created to test the efficiency of parallel computing from 3 different machine/OS with similiar number of core processors. The time taken to complete the number of iterations are noted in the log file output. User can set:

  • Number of rows when arranging the grid
  • Number of columns when arranging the grid
  • Number of iterations the base station should perform
  • Sea water column height threshold in metres
This project can be a good way to test the efficiency of parallel computing as 3 different machine/OS with similiar number of core processors were used to run the simulation. The time taken to complete the number of iterations are noted in the log file output.

Zombie Game

A 2D Zombie-themed game developed using Java language implementing object-oriented programming (OOP). The game was developed after designing it with a UML diagram. There are two maps where player can traverse between. There are weapons player can use to kill the zombie which roams around the map.

Each turn users are allow to select their action from a list provided. From moving & collecting food/weapons to attacking the zombies. There are trees and fence on the map which can be use to player advantage to beat the game. Player can also farm which will bear fruits after a few turns. Besides attacking the zombies physically, player can set trap to kill the zombies.

A boss, "Mambo Marie" will appear after a few turns with a higher max health compared to other relatives zombies.

Pong Game

A basic one-player pong game with a paddle movable by keyboard. The second paddle (right/ opponent) will follow the ball.

The ball will bounce of a random direction when hit by the paddle. Besides direction, the speed of the ball will also differs when hit at a certain angle.

To win, player will have to score 7 points against the opponent. Player will be able to restart after the game finishes.
